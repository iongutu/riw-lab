  <h2><a name="SECTION007470000000000000000">&nbsp;</a> <a name="l2h-260">&nbsp;</a> <br> 5.4.7 PythonHandlerModule </h2> 
  <p> <b><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Syntax" title="Syntax:">Syntax:</a></em></b> PythonHandlerModule module <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Context" title="Context:">Context:</a></em> server config, virtual host, directory, htaccess <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Override" title="Override:">Override:</a></em> not None <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Module" title="Module:">Module:</a></em> mod_python.c </p>
  <p> PythonHandlerModule can be used an alternative to Python*Handler directives. The module specified in this handler will be searched for existence of functions matching the default handler function names, and if a function is found, it will be executed. </p>
  <p> For example, instead of: </p>
  <div class="verbatim">
   <pre>
  PythonAuthenHandler mymodule
  PythonHandler mymodule
  PythonLogHandler mymodule
</pre>
  </div> 
  <p> one can simply say </p>
  <div class="verbatim">
   <pre>
  PythonHandlerModule mymodule
</pre>
  </div> 
  <p> </p>
