  <h2><a name="SECTION007210000000000000000">&nbsp;</a> <a name="l2h-251">&nbsp;</a> <br> 5.2.1 PythonInputFilter </h2> 
  <p> <b><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Syntax" title="Syntax:">Syntax:</a></em></b> PythonInputFilter handler name <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Context" title="Context:">Context:</a></em> server config <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Module" title="Module:">Module:</a></em> mod_python.c </p>
  <p> Registers an input filter <var>handler</var> under name <var>name</var>. <var>Handler</var> is a module name optionally followed <code>::</code> and a callable object name. If callable object name is omitted, it will default to "<tt class="samp">inputfilter</tt>". <var>Name</var> is the name under which the filter is registered, by convention filter names are usually in all caps. </p>
  <p> To activate the filter, use the <code>AddInputFilter</code> directive. </p>
