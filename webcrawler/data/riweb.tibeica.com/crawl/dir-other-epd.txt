  <h2><a name="SECTION007410000000000000000">&nbsp;</a> <a name="l2h-254">&nbsp;</a> <br> 5.4.1 PythonEnablePdb </h2> 
  <p> <b><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Syntax" title="Syntax:">Syntax:</a></em></b> PythonEnablePdb {On, Off} <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Default" title="Default:">Default:</a></em> PythonEnablePdb Off <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Context" title="Context:">Context:</a></em> server config, virtual host, directory, htaccess <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Override" title="Override:">Override:</a></em> not None <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Module" title="Module:">Module:</a></em> mod_python.c </p>
  <p> When On, mod_python will execute the handler functions within the Python debugger pdb using the <code>pdb.runcall()</code> function. </p>
  <p> Because pdb is an interactive tool, start httpd from the command line with the -DONE_PROCESS option when using this directive. As soon as your handler code is entered, you will see a Pdb prompt allowing you to step through the code and examine variables. </p>
