  <h2><a name="SECTION006820000000000000000">&nbsp;</a> <br> 4.8.2 Examples </h2> The following example demonstrates a simple hit counter. 
  <p> </p>
  <div class="verbatim">
   <pre>
from mod_python import Session

def handler(req):
    session = Session.Session(req)

    try:
        session['hits'] += 1
    except:
        session['hits'] = 1

    session.save()

    req.content_type = 'text/plain'
    req.write('Hits: %d\n' % session['hits'])
    return apache.OK
</pre>
