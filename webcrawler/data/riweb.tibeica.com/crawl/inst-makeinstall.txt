  <h2><a name="SECTION004310000000000000000">&nbsp;</a> <br> 2.3.1 Running make install </h2> 
  <p> </p>
  <ul> 
   <li>This part of the installation needs to be done as root. 
    <div class="verbatim">
     <pre>
    $ su
    # make install
</pre>
    </div> <p> </p>
    <ul> 
     <li>This will simply copy the library into your Apache <span class="file">libexec</span> directory, where all the other modules are. <p> </p></li> 
     <li>Lastly, it will install the Python libraries in <span class="file">site-packages</span> and compile them. <p> </p></li> 
    </ul> <p> <a name="l2h-16">&nbsp;</a> <a name="l2h-17">&nbsp;</a> <b>NB:</b> If you wish to selectively install just the Python libraries or the DSO (which may not always require superuser privileges), you can use the following <b class="program">make</b> targets: <b class="programopt">install_py_lib</b> and <b class="programopt">install_dso</b> </p><p> </p></li> 
  </ul>
