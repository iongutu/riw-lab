  <h2><a name="SECTION008110000000000000000">&nbsp;</a> <br> 6.1.1 Introduction </h2> 
  <p> To use the handler, you need the following lines in your configuration </p>
  <div class="verbatim">
   <pre>
  &lt;Directory /some/path&gt;
    SetHandler mod_python 
    PythonHandler mod_python.publisher
    &lt;/Directory&gt;
</pre>
  </div> 
  <p> This handler allows access to functions and variables within a module via URL's. For example, if you have the following module, called <span class="file">hello.py</span>: </p>
  <p> </p>
  <div class="verbatim">
   <pre>
  """ Publisher example """

  def say(req, what="NOTHING"):
  return "I am saying %s" % what
</pre>
  </div> 
  <p> A URL <code>http://www.mysite.com/hello.py/say</code> would return "<tt class="samp">I am saying NOTHING</tt>". A URL <code>http://www.mysite.com/hello.py/say?what=hello</code> would return "<tt class="samp">I am saying hello</tt>". </p>
  <p> </p>
