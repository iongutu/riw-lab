  <h2><a name="SECTION007310000000000000000">&nbsp;</a> <a name="l2h-253">&nbsp;</a> <br> 5.3.1 PythonConnectionHandler </h2> 
  <p> <b><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Syntax" title="Syntax:">Syntax:</a></em></b> PythonConnectionHandler handler <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Context" title="Context:">Context:</a></em> server config <br><em class="citetitle"><a href="http://httpd.apache.org/docs-2.0/mod/directive-dict.html#Module" title="Module:">Module:</a></em> mod_python.c </p>
  <p> Specifies that the connection should be handled with <var>handler</var> connection handler. <var>Handler</var> will be passed a single argument - the connection object. </p>
  <p> <var>Handler</var> is a module name optionally followed <code>::</code> and a callable object name. If callable object name is omitted, it will default to "<tt class="samp">connectionhandler</tt>". </p>
