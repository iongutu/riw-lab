  <h3><a name="SECTION008121000000000000000">&nbsp;</a> <br> 6.1.2.1 Traversal </h3> 
  <p> The Publisher handler locates and imports the module specified in the URI. The module location is determined from the <tt class="class">req.filename</tt> attribute. Before importing, the file extension, if any, is discarded. </p>
  <p> If <tt class="class">req.filename</tt> is empty, the module name defaults to "<tt class="samp">index</tt>". </p>
  <p> Once module is imported, the remaining part of the URI up to the beginning of any query data (a.k.a. PATH_INFO) is used to find an object within the module. The Publisher handler <i class="dfn">traverses</i> the path, one element at a time from left to right, mapping the elements to Python object within the module. </p>
  <p> If no path_info was given in the URL, the Publisher handler will use the default value of "<tt class="samp">index</tt>". If the last element is an object inside a module, and the one immediately preceding it is a directory (i.e. no module name is given), then the module name will also default to "<tt class="samp">index</tt>". </p>
  <p> The traversal will stop and <tt class="constant">HTTP_NOT_FOUND</tt> will be returned to the client if: </p>
  <p> </p>
  <ul> 
   <li>Any of the traversed object's names begin with an underscore ("<tt class="samp">_</tt>"). Use underscores to protect objects that should not be accessible from the web. <p> </p></li> 
   <li>A module is encountered. Published objects cannot be modules for security reasons. <p> </p></li> 
  </ul> 
  <p> If an object in the path could not be found, <tt class="constant">HTTP_NOT_FOUND</tt> is returned to the client. </p>
  <p> For example, given the following configuration: </p>
  <p> </p>
  <div class="verbatim">
   <pre>
  DocumentRoot /some/dir

  &lt;Directory /some/dir&gt;
  SetHandler mod_python
  PythonHandler mod_python.publisher
  &lt;/Directory&gt;
</pre>
  </div> 
  <p> And the following <span class="file">/some/dir/index.py</span> file: </p>
  <p> </p>
  <div class="verbatim">
   <pre>
  def index(req):

  return "We are in index()"

  def hello(req):

  return "We are in hello()"
</pre>
  </div> 
  <p> Then: </p>
  <p> http://www.somehost/index/index will return "<tt class="samp">We are in index()</tt>" </p>
  <p> http://www.somehost/index/ will return "<tt class="samp">We are in index()</tt>" </p>
  <p> http://www.somehost/index/hello will return "<tt class="samp">We are in hello()</tt>" </p>
  <p> http://www.somehost/hello will return "<tt class="samp">We are in hello()</tt>" </p>
  <p> http://www.somehost/spam will return "<tt class="samp">404 Not Found</tt>" </p>
  <p> </p>
